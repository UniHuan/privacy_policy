隐私政策

生效日期：2026年6月16日 · 版本 1.0

诗韵（以下简称"本应用"）高度重视用户隐私保护。本隐私政策旨在向您清晰、透明地说明本应用如何处理您的信息。本应用坚持数据最小化原则 — 能不收集的绝不收集，能本地存储的绝不上传。

一

我们收集的信息

本应用无需注册、无需登录，您可以直接使用全部功能。我们仅在以下有限场景中处理数据：

  • 收藏数据：您收藏的诗词标题及收藏时间，仅存储在您设备的本地 SwiftData 数据库中，不会上传至任何服务器。
  • 学习记录：答题分数、游戏成绩、打卡天数等统计数据，仅存储在您设备的 UserDefaults 中，不会上传。
  • AI 服务请求：当您使用「AI 深度赏析」或「AI 创作」功能时，本应用会将您当前查看的诗词内容或您输入的文本发送至第三方 AI 服务商（DeepSeek）的 API 接口进行处理。传输过程使用 HTTPS 加密。
二

我们不收集的信息

为最大程度保护您的隐私，本应用绝不收集以下任何信息：

  • 姓名、手机号、身份证号、电子邮箱等个人身份信息
  • GPS 定位、WiFi 信息等地理位置数据
  • 通讯录、相册、日历、健康等系统权限数据
  • 设备标识符（IDFA / IDFV / UDID）
  • 浏览历史、使用时长等行为分析数据

本应用不嵌入任何第三方分析 SDK、广告 SDK、统计 SDK 或用户追踪器。

三

信息使用与存储

本应用收集的有限信息仅用于以下目的：

  • 收藏列表的展示与管理
  • 学习统计（打卡天数、答题得分、游戏成绩）的个人展示
  • AI 功能：向 DeepSeek API 发送必要文本以生成赏析或创作内容

所有数据仅存储在您的 iOS 设备本地，受系统级数据保护机制和沙盒隔离保护。本应用不设后端服务器，不存在数据上传、云端同步或跨设备共享。

四

数据安全

我们采取合理的技术措施保护您的数据安全：

  • 本地数据受 iOS 系统级沙盒隔离保护，其他应用无法访问
  • AI 服务通信全程使用 HTTPS/TLS 加密传输
  • AI 服务请求仅包含当前操作的诗词文本，不含任何设备或用户标识信息
  • 用户输入在发送前经过去除控制字符、长度限制等安全处理

您可以通过以下方式彻底删除所有数据：卸载本应用，或在应用内取消所有收藏、清除学习记录。

五

第三方服务

本应用的 AI 功能（深度赏析、诗词创作）依赖DeepSeek API 提供技术支持。当您使用这些功能时，您当前操作的诗词文本或输入内容将通过 HTTPS 加密传输至 DeepSeek 服务器。

DeepSeek 的隐私政策请参阅：https://platform.deepseek.com/privacy

除 DeepSeek API 外，本应用不使用任何其他第三方服务。

六

儿童隐私

本应用的内容为中华传统诗词，适合所有年龄段的用户。本应用不收集任何用户的个人身份信息，因此不会有意收集或存储 13 岁以下儿童的个人数据。

如果您是家长或监护人，发现您的孩子在使用本应用过程中提供了个人信息（尽管本应用不收集此类信息），请通过下方联系方式告知我们，我们将及时处理。

七

您的权利

根据《中华人民共和国个人信息保护法》及相关法律法规，您享有以下权利：

  • 知情权：了解您的信息如何被处理（本政策即为行使此权利的载体）
  • 查阅权：在应用内直接查看您的收藏列表和学习统计数据
  • 删除权：随时取消收藏或卸载应用即删除所有本地数据
  • 注销权：本应用无需注册，因此不存在注销流程
八

本政策的更新

我们可能会适时更新本隐私政策。如有重大变更，我们将在应用内以适当方式通知您。更新后的政策将在本页面发布，并注明生效日期。

建议您定期查阅本政策，以了解最新的隐私保护信息。
