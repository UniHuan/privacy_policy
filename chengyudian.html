隐私政策

生效日期：2026年6月15日

一、信息收集

本应用坚持数据最小化原则，收集的信息严格限于提供服务所必需：

  • 收藏数据：您收藏的成语列表，存储于设备本地 SwiftData 数据库
  • 学习记录：打卡天数、答题分数、接龙成绩，存储于 UserDefaults
  • AI 问答：使用 AI 造句时，仅传输当前成语文本至第三方 API

本应用无需注册登录，不收集姓名、手机号、位置、相册等个人信息。

二、不收集的信息

  • 姓名、手机号、身份证号等个人身份信息
  • GPS / WiFi 定位数据
  • 通讯录、相册、日历数据
  • 设备标识符（IDFA / IDFV）
  • 浏览历史、点击行为等使用数据

三、信息使用

收藏数据仅用于展示您的成语收藏列表。学习记录用于统计打卡天数。AI 造句仅传输当前成语文本至第三方服务。

四、存储与安全

所有数据存储在 iOS 设备本地，由 SwiftData 框架管理，受系统级数据保护和沙盒隔离保护。AI 造句通过 HTTPS 加密传输。

卸载应用即删除所有本地数据。

五、用户权利

根据《个人信息保护法》，您享有：知情权（本政策说明）、查阅权（收藏列表）、更正权（取消/重新收藏）、删除权（卸载即删除）、注销权（无需注册卸载即视为注销）。
